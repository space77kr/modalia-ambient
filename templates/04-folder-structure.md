# Standard Song Folder Structure

Use this structure for every new song project.

```
song-name/
│
├── README.md
├── lyrics.md
├── suno-prompt.md
├── production-notes.md
├── versions.md
│
├── audio/
│   ├── drafts/
│   └── selected/
│
├── cover/
│   ├── concept.md
│   └── image-prompt.md
│
└── release/
    ├── metadata.md
    ├── youtube.md
    ├── spotify.md
    └── hashtags.md
```

## Naming Rules

- Song folders use lowercase.
- Separate words with hyphens.
- Number songs sequentially.

Example:

```
01-blue-skies
02-summer-breeze
03-golden-hour
04-midnight-rain
```

## Goal

Every song follows exactly the same structure.
